Com variaveis aprendemos que é possivel guardar dados , mas se eu quiser guardar mais de um dado em uma variavel ?
É possivel , porem como me refiro aos dados armazenados ? , dado1 , dado2 .... É preciso usar coleçoes

Listas

Lista = [ "dado0" , "dado1" , "dado2" ]

Nomeia a variavel de lista , para ficar mais facil entendimento , quando for criar uma lista é necessario usar os colchetes [ ] ,dentro dos colchetes é possivel armazenar numeros ou strings

Numeros = [ 1 , 2 , 3 ]

Lista é uma variavel que possui mais de um dado , mais de 1 item , e para acessa - lo basta usar o seu numero de id , em coleçoes cada item possui seu id para que possa se referir a aquele dado .
Usando exemplo

Numeros = [ 1, 2 , 3 ]

Os id começam pelo numero 0 e vai assim por diante dependendo do numero de itens na lista , para acessar um dado expecifico com o numero 3

Print(Numeros[2])

mostre o item do id correspondente a 2

Resultado

3

Nome dado a lista e o uso de colchetes com o numero de id do item,ja que o id começa do 0 o item de id 3 é o 2

Itens - 1 2 3
Id - 0 1 2

Existem mais ferramentas , funções , como adicionar , remover itens de listas e etc . Porém o foco é entender o conceito, ao precisar de uma ferramenta necessaria procure - a.

Tuplas são assim como as listas a diferença é que a tupla tem o paranteses presentes na estrutura

Numeros = (1, 2 , 3 )

Para referir ao id de um item assim como na lista e no dicionario que aparecerá posteriormente , Todos usam o colchetes para se referir ao [ id ]

Print(Numeros[2])

Entao por que usar tuplas ? As tuplas não podem ser alteradas , são imutaveis usando funções especificas como adicionar item e remover , não funcionam em tuplas dependendo da a
situaçao tuplas podem ser usadas

Dicionarios

Listas usa - se - [ ]

Tuplas usa - se - ( )

Dicionarios usa - se - { } chaves, keys

Como dito antes dicionarios tambem usam se colchetes para se referir ao id porem ao diferencial do dicionario ao resto que é o que faz com que ele seja usado em ocasioes especias ,
Dicionarios possuem o id personalizados , para cada item você pode nomea - los , assim como uma variavel em que você a nomeia , e se referindo por esse nome se obtem o dado

Nesse caso ao nomear o id é por esse nome em que será referido a aquele item ,para usar o dicionarios

Dicionario = {" nome" : " roberto" }

Print(dicionario["nome"] )

Resultado

roberto
